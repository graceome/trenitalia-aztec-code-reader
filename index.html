Trenitalia Aztec Code Reader

This is an attempt to reverse engineer the content of AZTEC codes printed on Trenitalia tickets. The size of the binary code has evolved over the years. It used to be 440 bit long before 2012(?) and is now 536 bits. The current database of tickets is too small to understand what most of the descriptors are (their value is always the same) but it is rather clear that no cryptographic signatures or integrity checks are present. The database of known station names is also quite small (shown below). The position and size of the known fields (about half of the 536 bits) and their meaning is presented in the table below. Notably the year field is missing so the conversion from day of the year to date must be done by hand. If you have questions or you want to contribute to this project contact gracemomo@pm.me or open an issue on github.

This page also includes an interactive parser based on a slightly modified version of zxing-js (see footer). Everything is computed locally inside your browser and not data is shared.
